---
description: Автоматический мониторинг и исправление билдов Vercel (Build Sentinel)
---

Этот ворклист позволяет Антигравити самостоятельно обнаруживать упавшие билды, анализировать ошибки и деплоить фиксы без постоянного подтверждения команд.

// turbo
1. Проверить статус последнего деплоя:
   `vercel list --limit 1`

2. Если статус 'Error', получить детальные логи:
   `vercel inspect [deployment-url]`

3. Проанализировать текст ошибки (например, Missing Imports или Type Errors).

4. Выполнить автоматическое исправление в коде.

// turbo
5. Запустить локальную проверку типов (чтобы не спамить в сеть):
   `npm run build`

// turbo
6. Если билд успешен, зафиксировать изменения и отправить в репозиторий:
   `git add . ; git commit -m "fix: automated build repair by Build Sentinel" ; git push`

7. Подтвердить финальный статус деплоя.
